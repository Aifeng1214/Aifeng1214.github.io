深入理解Java的多态

发表于 2018-04-09 | 分类于 Java

Java作为一个面向对象的语言，封装、继承和多态是三大特性。封装和继承比较好理解，并且Java摈弃了C++中的多重继承也就更加方便理解了。这里帮助自己回顾一下对于多态的理解。

Java多态性

什么是Java的多态性，引用网上的一句总结就是：

事物在运行过程中存在不同的状态。

多态的存在有三个前提：

  • 要有继承关系
  • 子类要重写(overwrite)父类的方法
  • 父类引用指向子类对象
